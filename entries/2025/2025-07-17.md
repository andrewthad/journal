# Substrait Protobuf Woes

I figured out how to inspect a binary protobuf message produced by calling
`to_substrait` on a pyarrow `Expr`. Run this command:

    buf convert buf.build/substrait/substrait --type=substrait.ExtendedExpression --from=expr.binpb --to=expr.txtpb

If you try to turn it into json instead, it fails with:

    Failure: --to: json marshal: proto: google.protobuf.Any: unable to resolve "type.googleapis.com/google.protobuf.UInt64Value": not found

But turning it into text works for some reason.
